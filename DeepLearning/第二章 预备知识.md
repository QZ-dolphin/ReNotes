常用张量创建方法
```python
x = torch.arrange(4) # x = tensor([0, 1, 2, 3]) 从0开始的4个整数

y = torch.arrange(12)
y.reshape(3, 4) # 将原本形状为(12,)的行向量转换为形状为(3,4)的矩阵
# 可通过-1自动计算出只一个没有给出的维度大小
y.reshape(3, -1)
y.reshape(-1, 4)

# 创建全0张量
a = torch.zeros(2, 3, 4)

# 创建全1张量
a = torch.ones(2, 3, 4)

# 通过python列表转换
b = [1, 2, 3, 4]
b = torch.tensor(b)
```
