# C++ 笔记
## 预备知识
让运行终端窗口一直打开，直到按下任意键，在 `return` 语句前添加
```C++
cin.get();
```
`#include` 预处理器编译指令
`using namespace` 编译指令
函数头包含返回类型+函数名+形参列表
函数定义 包含 函数体

隐含的返回语句：仅在`main()`函数中，如果编译器到达`main()`函数末尾时没有遇到返回语句，则认为以`return 0;`结尾。

对象是类的特定实例，而类定义了数据的存储和使用方式。

`endl` 确保程序继续运行前刷新输出（将其立刻显示在屏幕上）；而使用`"\n"`不能提供这样的保证。

声明语句：定义声明、引用声明

可以连续使用赋值运算符
```c++
int a, b, c;
a = b = c = 3;
```

函数库、类库

函数原型之于函数就像变量声明之于变量，应在首次使用函数之前提供原型。原型结尾的分号表明它是一条语句，使得其为一个原型，而不是函数头。
